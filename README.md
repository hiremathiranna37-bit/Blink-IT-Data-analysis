# Blink-IT-Data-analysis
This project focuses on real-time sales analysis of the BlinkIT grocery dataset using Microsoft Excel. The objective is to analyze large sales datasets and present meaningful business insights through an interactive dashboard that supports data-driven decision-making.

Objectives
  
  - Perform end-to-end sales data analysis
  - Clean and transform large datasets for accuracy
  - Design an interactive dashboard to visualize key performance metrics
  - Generate actionable insights to improve business visibility and sales 
    
Tools
  - Microsoft Excel
